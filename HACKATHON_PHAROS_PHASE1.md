# Spesifikasi Teknis Skill (dari Agent Center) ✅

> Sumber: https://www.pharos.xyz/agent-center — **ini menjawab TODO teknis sebelumnya**

## Pharos Skill Engine

Skill resmi yang membundel kapabilitas onchain inti. Satu integrasi = full blockchain workflow.

- **GitHub:** https://github.com/PharosNetwork/pharos-skill-engine
- **Install:** `npx skills add https://github.com/PharosNetwork/pharos-skill-engine`
- Juga tersedia via: download zip / npm / pnpm / bun
- **Download zip (v0.1.0):** https://github.com/PharosNetwork/pharos-skill-engine/archive/refs/tags/v0.1.0.zip
- **Lisensi:** MIT-0 (bebas pakai, modif, redistribusi, tanpa atribusi)

## Framework yang Didukung

Agent Center mendukung beberapa Agent & Coding Framework: **OpenClaw, Claude Code, Codex**, dan lainnya. Ada built-in skill usage examples + prompts untuk quick start.

## Kapabilitas Bawaan (Skill Engine)

- Check Balance (native + ERC-20 via JSON-RPC)
- Check Transaction Status
- Read Contract (read-only methods)
- Send Transaction (transfer aset / invoke kontrak)
- Gas Estimation (simulasi tx)
- Deploy Contract (bytecode + ABI, EVM-compatible)
- Quick ERC20 deployment
- Batch transfer / airdrop
- Wallet asset aggregation
- Contract deployment & verification

## Contoh Prompt (Usage Examples dari Terminal)

1. **Query Balance:** "Query the USDC balance for address {addr} on the Pharos {network}. Show human-readable balance."
2. **Send Native Transfer:** "Send 0.0001 {native_token} from my wallet to {to_addr}. Show tx hash + confirmation status."
3. **Deploy Contract:** "Deploy a simple storage contract... store/retrieve a uint256. Use Solidity, deploy via forge."
4. **Batch Airdrop:** "Airdrop 1 USDC each to 3 addresses. USDC contract {token_addr} (decimals: 6)."
5. **Contract Read Total Supply:** "Call totalSupply() on USDC contract at {contract_addr}. Human-readable."
6. **Gas Estimation:** "Estimate gas to call transfer(address,uint256) sending 100 USDC to {to_addr}."
7. **Check Transaction Status:** "Query status of tx {tx_hash}. Show status, from, to, value, gas used."
