<!-- Cascade public API the Skill documents. Extracted from src/Cascade.sol — executor uses these directly, no contract exploration needed. -->

Live deployed instance (atlantic-testnet, chainId 688689):
  CASCADE = 0xd41C32562D0BE20D354120E1De11A91abC340F50

Write functions:
  register(uint256 price, uint256[] depIds, uint256[] depShares) returns (uint256 id)
    - price: native-token invoke price in wei (may be 0 for a leaf)
    - depIds: dependency skill ids; each must be a strictly-smaller, already-registered id
    - depShares: bps share (0-10000) routed into each dep's subtree; Σ depShares ≤ 10000
    - reverts: "len mismatch", "bad dep id", "shares > 10000", "depth > 8"
  invoke(uint256 skillId) payable
    - exact payment: msg.value MUST equal skills[skillId].price
    - reverts: "no skill", "wrong value"
  claim() returns (uint256 amount)
    - withdraws caller's accrued balance; zero-before-send (double-claim pays 0, no revert)

Read functions:
  balances(address) returns (uint256)   — accrued claimable balance per creator
  skillCount() returns (uint256)         — highest valid skill id

Events:
  SkillRegistered(uint256 indexed id, address indexed creator, uint256 price)
  Invoked(uint256 indexed skillId, address indexed payer, uint256 amount)
  RoyaltyAccrued(uint256 indexed skillId, address indexed creator, uint256 amount)
  Claimed(address indexed creator, uint256 amount)

Forge script env vars (from script/*.s.sol):
  PRIVATE_KEY  — signer (gitignored .env), never logged
  CASCADE      — deployed Cascade address
  PRICE        — register: invoke price in wei
  DEP_IDS      — register: comma-separated dep ids (optional, default empty)
  DEP_SHARES   — register: comma-separated bps shares, 1:1 with DEP_IDS (optional)
  SKILL_ID     — invoke: id of the skill to invoke
  INVOKE_VALUE — invoke: exact wei to send (must equal registered price)
  Network resolved at run time via --rpc-url (e.g. --rpc-url atlantic_testnet from foundry.toml [rpc_endpoints]).
</interfaces>
